# Client Desk — прототип CRM для юриста

Тестовое задание: таблица клиентов, статусы дел, счётчики.

**Live:** https://elenasamanchuk.github.io/lawyer-crm/

## Запуск

```bash
npm install
npm run dev
```

## Стек

React + Vite + TypeScript, данные в `localStorage`.

Собрано в Cursor: UI, логика статусов, адаптив, деплой.
