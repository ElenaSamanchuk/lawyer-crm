# LexDesk — CRM-прототип для юриста

Тестовое задание: клиенты, статусы дел, счётчики, валидация полей, напоминания.

**Live:** https://elenasamanchuk.github.io/lawyer-crm/

## Запуск

```bash
npm install
npm run dev
```

## Что реализовано

- Таблица клиентов с поиском, фильтром и сортировкой
- Добавление клиента: имя, телефон, тип дела, статус, дата следующего контакта
- Валидация полей (телефон в международном формате, дата не в прошлом)
- Смена статуса: Новый → В работе → Закрыт
- Счётчики по каждому статусу + просроченные контакты
- Напоминание клиенту: WhatsApp / Telegram с готовым шаблоном
- Браузерные уведомления при добавлении и смене статуса
- Подсветка просроченных дел и клиентов без контакта &gt; 3 дней

## Стек

React + Vite + TypeScript + Tailwind CSS, данные в `localStorage`.
